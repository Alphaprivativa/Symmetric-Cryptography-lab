## 📖 Description
This assignment focuses on stream cipher cryptanalysis using the **MAGMA** computer algebra system. You will implement core algorithms for analyzing Linear Feedback Shift Registers (LFSRs) and perform correlation attacks on combination generators. The tasks cover the Berlekamp-Massey algorithm, optimal initial state recovery, and a full divide-and-conquer correlation attack on two predefined combiner structures.

## 🛠️ Exercises

### Exercise 1: Berlekamp-Massey Algorithm
Implement the Berlekamp-Massey algorithm to determine the linear complexity and feedback polynomial of a binary sequence.
- `Berlekamp_Massey(s)`: Takes a sequence of bits `s`.
- **Output:** `l` (integer, linear complexity) and `f` (polynomial over GF(2), feedback polynomial of the minimal LFSR).
- **⚠️ Important:** You **cannot** use MAGMA's built-in `BerlekampMassey` function. Implementation must be from scratch.

### Exercise 2: Initial State Recovery
Find the initial states of a set of LFSRs that maximize the correlation with a given output sequence.
- `InitialStates(s, Pol)`: Takes a bit sequence `s` and a list of feedback polynomials `Pol = [f1, ..., ft]`.
- **Output:** A list of bit sequences `L = [st1, ..., stt]` representing the initial states that maximize the correlation $1 - d_H(s, \sigma)/N$, where $\sigma$ is the XOR sum of the individual LFSR outputs generated with those initial states.

### Exercise 3: Correlation Attack on Combiner Generators
Implement a procedure that performs a correlation attack to recover the initial states of two combination generators from known keystream data.
- `CorrelationAttack(~L)`: Takes an empty list `L` (passed by reference via `~`).
- **Input Data:** Automatically reads `dataSeq.mag` from the working directory, which contains two keystream sequences `s1` and `s2`.
  - `s1` is generated by **4 LFSRs** (specific feedback polynomials provided in the PDF) combined with the Boolean function $f = x_1x_2 + x_2x_3 + x_2x_4 + x_3$.
  - `s2` is generated by **3 LFSRs** (specific polynomials provided) combined with $f = x_1x_2 + x_1x_3 + x_2x_3 + x_1 + x_2 + 1$.
- **Output:** Updates `L` to `[L1, L2]`, where `L1` contains the 4 recovered initial states for `s1` and `L2` contains the 3 recovered initial states for `s2`.
- **Requirement:** The procedure must be **well-commented in English**, clearly explaining and motivating the cryptanalytic steps (e.g., correlation measurement, exhaustive search per register, thresholding).
